# 幼儿园幼师PPT制作助手

专为幼儿园教师设计的PPT制作工具，提供专业的模板、内容框架和素材资源。

## 核心功能

### 1. PPT模板生成
- 年终工作总结汇报
- 寒暑假安全注意事项
- 家长会演示文稿
- 教学计划展示
- 节日活动策划

### 2. 内容框架
- 专业的幼教内容结构
- 符合幼师工作特点的汇报逻辑
- 家长关心的重点内容
- 教育局要求的标准格式

### 3. 素材资源
- 可爱的卡通图标
- 幼儿园主题背景
- 教育相关图表模板
- 节日装饰元素

## 常用PPT类型

### 工作汇报类
使用 `scripts/work_report_generator.py` 生成工作总结PPT。

### 安全教育类  
使用 `scripts/safety_education_generator.py` 生成安全教育PPT。

### 家长会类
使用 `scripts/parent_meeting_generator.py` 生成家长会PPT。

### 教学计划类
使用 `scripts/teaching_plan_generator.py` 生成教学计划PPT。
